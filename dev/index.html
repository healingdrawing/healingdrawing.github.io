  • nvm - node.js version manager
  • npm - node.js package manager
  • node.js - open-source, cross-platform environment to run javascript on server side.
  • TypeScript - superset of javascript that adds static types.
  • Vite - frontend build tool and development server. Supports typescript.
  • Babylon.js - 3d open-source web rendering engine. Supports typescript + vite.
  • Vue.js - performant and versatile framework for building web user interfaces. Supports typescript.
