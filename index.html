Kotlin Code Scroller 2018


Create and display HTML page with Kotlin code, highlighted uses highlight.js.
It will be comfort way to displaying code with smooth scrolling, inside your video tutorials.
Line numbers can be displayed.

 

Human Dummy Creator 2018


Create human styled dummy for 2.5D animation sketch.
Can be comfort to keep the proportions of animation frames.
Result can be saved/loaded uses txt file, exported into OBJ file (that later be imported into other 3D tools), rendered as PNG image.

Available presets:
-configurable size of dummy (limbs length, volume of body parts);
-configurable dummy pose (limbs relative bend angles, back bend angles - vertebra's pose);
-switching pose between left and right;
-copying pose from left/right side to another;
-reset dummy pose to default strait position;
-markers of limbs end projection and markers of center projection of dummy body;
-global circle to keep scale in time of animation creation(when you export/import PNG in gimp/incskape etc);
-configurable dummy parts colors;
-configurable background color;
-configurable three lights (ambient, directional, point);
-configurable camera (distance, oy angle, oz angle).

 

Gradient Generator 2018


Generate gradients for GIMP (ggr) and inkscape (svg), and save it that use later.
Optimized for mobile vertical screen orientation.
That save result for later editing, press "save seed", then later press "load seed" and play with "random color" or "random size".
Application was created that make process of drawing planetary rings faster.

Available presets:
-center hole radius(for using as radial);
-number of color steps;
-limit alpha of steps color
-separately limit color R G B components;
-RGB or monochrome mode;
-few modes to limit color(same as color palette), limit size, limit transparency(limit alpha);
-few generator mode to randomize color, size, all parameters
-transparent or smooth connected hole colors;
-can be saved as html file and used without internet connection.

 

Rocket Creator 2018


Create a parametric 3D model of rocket, bullet, space ship dummy.
Result can be saved/loaded uses txt file, exported into OBJ file (that later be imported into other 3D tools), rendered as PNG image.

Available presets:
-configurable random rocket generator;
-configurable base form;
-configurable tail form;
-configurable tail ring form;
-configurable tail bottle form;
-configurable colors of rocket elements;
-configurable three lights(ambient, directional, point);
-render scene into PNG file;
-wireframe model displaying;
-can be saved as html file and used without internet connection.
